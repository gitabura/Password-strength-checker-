🔐 Password Strength Checker

A modern and responsive web application that evaluates the strength of a user's password in real time. The application helps users create secure passwords by providing instant feedback based on widely accepted password security practices.

---

📖 About the Project

Passwords are the first line of defense against unauthorized access. Weak passwords make accounts vulnerable to cyberattacks such as brute-force and dictionary attacks.

The Password Strength Checker analyzes a password as the user types and categorizes it as Weak, Medium, or Strong based on several security criteria. It is designed to improve cybersecurity awareness while demonstrating front-end web development skills.

⚙️ How It Works

The application evaluates the password using common security rules:

- Password length
- Presence of uppercase letters (A–Z)
- Presence of lowercase letters (a–z)
- Presence of numbers (0–9)
- Presence of special characters (!@#$%^&*)

Based on these checks, the password is classified as:

- 🔴 Weak
- 🟡 Medium
- 🟢 Strong

The result updates instantly as the user types.

🔒 Password Security Tips

- Use at least 8–12 characters.
- Include uppercase and lowercase letters.
- Add numbers and special characters.
- Avoid common words and personal information.
- Use a unique password for every account.
- Consider using a password manager.
